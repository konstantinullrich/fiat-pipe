# Fiat Pipe
A REST-API to load Fiat pricing from CoinGecko anonymously.

This Project is built for the Oxen Wallet.  

## Get started

1. Install dependencies
    ```
    $ pip install -r requirements.txt
    ```

2. Run the Server
    ```
    $ python fiatpipe.py
    ```
   
3. Be happy
